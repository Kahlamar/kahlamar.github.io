# Portfolio - Cyril | Testeur Junior QA 🚀

Bienvenue sur le dépôt de mon portfolio personnel ! Ce projet web présente mon profil hybride, combinant mon background de développeur Fullstack (Angular, .NET) avec ma nouvelle orientation vers l'Assurance Qualité (QA) et le Testing Logiciel.

## 🛠️ Technologies Utilisées

Ce projet est un site statique "vanilla" construit sans framework lourd, misant sur la légèreté et la performance :

- **HTML5** : Structure sémantique et bonnes pratiques d'accessibilité (SEO, balises alt, hiérarchie des titres).
- **CSS3** : Mise en forme personnalisée, gestion des contrastes et effets visuels (transitions, gradients).
- **JavaScript (ES6)** : Génération dynamique du contenu (compétences et projets) via l'API `fetch` et manipulation du DOM.
- **Bootstrap 5** : Système de grille responsif et composants pré-stylisés (Navbar dynamique, Cards).
- **FontAwesome** : Intégration d'icônes vectorielles pour la section contact.

## ✨ Fonctionnalités Principales

- **Design Responsif** : S'adapte parfaitement aux écrans d'ordinateurs, tablettes et smartphones.
- **Navbar Dynamique** : Barre de navigation transparente au sommet qui s'assombrit au défilement pour une meilleure lisibilité.
- **Génération Dynamique via JSON** : Les cartes de la section "Compétences" et "Portfolio" sont générées automatiquement à partir de fichiers locaux (`skills.json`, etc.), rendant les mises à jour futures extrêmement simples.
- **Optimisation SEO & Accessibilité** : Respect des standards Lighthouse et WAVE WebAIM (contrastes validés, hiérarchie de titres stricte).

## 🚀 Installation et Utilisation

Pour lancer ce projet localement :

1.  Clonez ce dépôt :
    ```bash
    git clone [https://github.com/Kahlamar/OC_Portfolio](https://github.com/Kahlamar/OC_Portfolio)
    ```
2.  Ouvrez le dossier dans votre éditeur de code (ex: Visual Studio Code).
3.  Pour que le chargement du fichier JSON fonctionne correctement (en raison des politiques CORS des navigateurs), utilisez une extension comme **Live Server** sur VS Code pour lancer le projet, ou hébergez-le sur un serveur local.
4.  Ouvrez `index.html` dans votre navigateur.
